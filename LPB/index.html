Langage de Programmation

Connaître les concepts, les niveaux, les paradigmes, la syntaxe générale, les forces et les faiblesses des langages couramment utilisés dans les applications web.

Introduction à la programmation : logique de programmation, concepts de base de la programmation procédurale, éléments/outils.

Introduction aux aides à la programmation : extension des fonctionnalités d’un langage par les moteurs, les librairies et les frameworks.

L’évaluation se fera au moyen d’exercices pratiques journaliers.
